🚀 Built an End-to-End NLP Pipeline for Amazon Review Sentiment Analysis

I recently worked on a project where I transformed raw Amazon reviews into meaningful sentiment insights using Machine Learning.

🔍 Key Highlights:

* Web scraping with retry & session handling
* Advanced text preprocessing (including negation handling)
* Feature engineering using TF-IDF & CountVectorizer
* Model training with Logistic Regression, Naive Bayes & SVM
* Cross-validation based model selection
* Deployment-ready models using Pickle

📊 This project helped me deeply understand how text encoding impacts model performance and how to build a production-ready NLP pipeline.
